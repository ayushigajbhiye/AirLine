#  Flight Go - Airline Reservation Management System

##  Overview
Flight Go is a Java-based Airline Reservation Management System developed using JDBC and MySQL.  
The project allows users to search flights, book tickets, make payments, and manage bookings, while admins can manage flights and view reports.

This project demonstrates:
- Core Java
- JDBC Connectivity
- MySQL Database Operations
- SQL Queries
- Stored Procedures and Functions
- CRUD Operations

# ⚙️ Setup Instructions

## 1️ Clone Repository
```bash
git clone https://github.com/your-username/flight-go.git
```

---

## 2️ Configure Database

Create the database in MySQL:

```sql
CREATE DATABASE airlines;
```

Update database credentials in `Main.java`:

```java
static final String JDBC_URL = "jdbc:mysql://localhost:3306/airlines";
static final String User = "root";
static final String password = "root";
```

---

## 3️ Add MySQL JDBC Driver

Download and add MySQL Connector/J library to your project.

---

## 4️ Run the Project

Compile and run the project:

```bash
javac Main.java
java Main
```
